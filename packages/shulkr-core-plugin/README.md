# shulkr-core plugin

Companion Paper plugin for the [shulkr](https://shulkr.dev) panel. Its **only** purpose is to
push verified real-time telemetry to the shulkr backend so the Analytics page can display
accurate data.

**Parity rule:** shulkr must behave identically whether this plugin is installed or not.
The plugin is purely additive — it never replaces any panel mechanism.

## Build

```bash
./gradlew build
```

Produces `build/libs/shulkr-core-<version>.jar` and copies it into
`packages/backend/dist/assets/plugins/shulkr-core.jar` (consumed by the backend installer).

## Manual install

Drop `shulkr-core.jar` into `plugins/` on your Paper server and create
`plugins/shulkr-core/config.yml` with:

```yaml
backend_url: http://your-shulkr-host:3000
server_id: <id shown by the shulkr panel>
token: <token generated by the shulkr panel>
push_interval_seconds: 5
protocol_version: 1
```

Then restart the Minecraft server. Use `/shulkr status` to verify the connection and
`/shulkr reload` after updating the config.

In normal use, prefer the "Install" button in the shulkr panel which handles the copy
and config generation for you.
